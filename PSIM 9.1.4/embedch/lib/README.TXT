/# History: created by Harry H. Cheng, 2/13/1994

The following subdirectoris are contained in this directory:

libc     --- function files for standard C/POSIX libraries
libch    --- general-purpose function files
libopt   --- lib of functions not ubiquitously available in all platforms.
             These functions are available only to regular shell. 
             These function will not be included if Ch is executed with option -a
